# AGPK package sources (official)

Open package **source trees** for AgentNet AGPK market packages live **here** under `AIAGENTNETLLC/AGPK-MARKET`.

| Path | Role |
|------|------|
| `packages/org.forgejo.Forgejo.core` | Forgejo service core (binary+systemd+PG) |
| `packages/org.forgejo.Forgejo.ui` | Forgejo desktop workbench UI |

- **Releases** = installable tar.gz + sha256 (share discovery)
- **This tree** = human-readable / agent-readable package source (not the private monorepo)

Do **not** store open AGPK package sources in personal private repos.
